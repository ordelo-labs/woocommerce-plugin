Integração completa entre WooCommerce e Mercado Livre via Ordelo. Sincronize produtos e gerencie pedidos automaticamente em tempo real.

== Description ==

Conecte sua loja WooCommerce ao maior marketplace da América Latina de forma simples e eficiente.

Este plugin oferece integração completa com o Mercado Livre através da plataforma Ordelo, permitindo:

* **Sincronização automática de produtos** - Seus produtos são enviados automaticamente para o Mercado Livre
* **Atualização em tempo real** - Preços, estoque e informações são sincronizados instantaneamente
* **Gestão centralizada de pedidos** - Gerencie todos os pedidos diretamente do seu WooCommerce
* **Configuração simplificada** - Interface intuitiva para conectar sua conta Ordelo

Ideal para lojistas que desejam expandir suas vendas no Mercado Livre sem complicações técnicas.

== Installation ==

= Instalação Manual =

1. Faça download do arquivo .zip do plugin
2. No painel do WordPress, vá em "Plugins" > "Adicionar novo" > "Enviar plugin"
3. Selecione o arquivo .zip e clique em "Instalar agora"
4. Ative o plugin após a instalação

= Configuração Inicial =

1. Após ativar o plugin, vá em "WooCommerce" > "Configurações" > "Token de integração ordelo"
2. Insira suas credenciais da conta Ordelo:
   * API Key
5. Salve as configurações
